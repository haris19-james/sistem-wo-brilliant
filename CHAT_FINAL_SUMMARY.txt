💡 KEY TECHNICAL IMPLEMENTATIONS:

1. Message Sending Flow:
   ┌─ JavaScript captures textarea value
   ├─ AJAX POST /lapangan/chat/send
   ├─ ChatController validates & creates record
   ├─ Database INSERT chat_messages
   ├─ Return JSON response
   └─ JavaScript appends bubble + scroll

2. Message Receiving Flow:
   ┌─ Admin sends message (other panel)
   ├─ Database records: is_read = false
   ├─ Korlap dashboard polls (or opens chat)
   ├─ getLatestConversations() finds new message
   ├─ Unread badge displays count
   ├─ Click conversation triggers getConversation()
   ├─ Database UPDATE is_read = true
   └─ Badge disappears on next refresh

3. Avatar Initials Generation:
   "Adi Dharmawan" → Take first letter of each word → "AD"
   Handles: Single name, null names, special characters

4. Smart Timestamp Formatting:
   Today: 09:15 (HH:mm format)
   Yesterday: "Kemarin"
   < 7 days: "X Hari lalu"
   Older: "d M" (25 May)

5. Query Optimization:
   Instead of:     SELECT * FROM chat_messages WHERE ...
   Use:            SELECT DISTINCT contact_id, MAX(id) ...
   Benefit:        ~50% faster for large datasets
