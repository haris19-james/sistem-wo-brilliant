# 💍 Sistem Informasi Wedding Organizer

Sistem Informasi Wedding Organizer adalah aplikasi berbasis web yang dirancang untuk membantu pengelolaan layanan pernikahan secara digital. Aplikasi ini memudahkan pelanggan dalam melihat paket pernikahan, melakukan pemesanan, dan memantau status layanan, serta membantu admin dalam mengelola data pelanggan, paket, transaksi, dan laporan.

## 🚀 Fitur Utama

* Manajemen data pelanggan
* Manajemen paket wedding
* Pemesanan paket secara online
* Pengelolaan transaksi pembayaran
* Dashboard admin
* Laporan pemesanan
* Sistem autentikasi login

## 🛠️ Teknologi yang Digunakan

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* Bootstrap
* Laragon/XAMPP

## ⚙️ Instalasi

1. Clone repository

```bash
git clone https://github.com/haris19-james/sistem-wo-brilliant.git
```

2. Pindahkan project ke folder web server

```text
C:\laragon\www\
```

atau

```text
C:\xampp\htdocs\
```

3. Import database MySQL.

4. Konfigurasi koneksi database pada file konfigurasi.

5. Jalankan web server dan akses melalui browser.
